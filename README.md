# Patient Management API

A CRUD REST API built using FastAPI and Pydantic.

## Features

- Create Patient
- View All Patients
- View Patient by ID
- Update Patient
- Delete Patient
- Query Filters
- Pydantic Validation
- Swagger Documentation

## Tech Stack

- Python
- FastAPI
- Pydantic
- JSON

## Run Locally

```bash
pip install fastapi uvicorn pydantic
python -m uvicorn Fast_API:app --reload
```

Open Swagger Docs:

```text
[http://127.0.0.1:8000/docs](https://patient-management-api-fastapi-2.onrender.com/)
```
